HTML

  • The head element contains information about the wbpage.
  • The body element represents the visible content shown to the user.
  • The footer contains author information, copyright information,carriers.
  • The title element defines the title for the webpage.
  • The Style element defines CSS styles asscociated with a website.
  • The Base element defines the base URL for a webpage.
  • The Link element connects an external resource to the HTML document.
  • The Meta element defines metadata such as the character set, description, keywords, auther, and viewport.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The * is a special wild card symbol to declare that all elements on the page will have the style properties.
  • The img selector assigns property value to any image element in the markup.
  • Two or more selectors can be combined with a comma [ , ] allowing the declartions to affect both selectors.
  • The class selector selects elements with a specific class attriubte. Example [ .nameofclass {} ].

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone : copy and/or paste directory.
  • git add : moves changes from the working directroy to the staging area.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Console.log outputs the code within the script.
